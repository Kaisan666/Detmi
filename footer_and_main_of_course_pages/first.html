Курс по Python

Что вас ждет?

Данный курс является вводным по языку Python, в нем рассматриваются базовые структуры, после него у вас будет сильная база по языку. После каждой темы идут практические задания.

Для кого подойдет данный курс?

Данный курс подойдет для тех, кто еще совсем не знаком с данным языком программирования, но имеет огромное желание изучить его.
Данный курс станет для него проводником.
